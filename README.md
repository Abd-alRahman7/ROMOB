# Components
- PIC16F877A microcontroller
- Ultrasonic Distance Sensor
- Servo Motor
- 2 DC Motors
- Temperature Sensor
- Fan
- Motor Driver (H-Bridge)
- CRYSTAL Oscillator (8 MHz)
- 3 lithium-ion Battaries
